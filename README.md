# Expense Tracker

A simple Expense Tracker web application built using HTML, CSS, and JavaScript. It allows users to add expenses, track total spending, and store data using Local Storage so expenses remain available even after refreshing the page.

## Features

* Add new expenses with a name and amount
* Automatically calculate total expenses
* Delete individual expenses
* Data persistence using Local Storage
* Responsive and beginner-friendly project

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## How It Works

1. Enter an expense name and amount.
2. Click the Add Expense button.
3. The expense is added to the list.
4. The total amount updates automatically.
5. Expenses are saved in Local Storage.
6. Users can remove expenses using the Delete button.
